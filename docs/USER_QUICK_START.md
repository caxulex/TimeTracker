## â“ Common Questions

### Q: What if I forgot to start my timer?

Add a **manual entry** with the correct start and end times.

### Q: Can I track multiple tasks at once?

No, only one timer can run at a time. Complete one task before starting another.

### Q: How do I see my team's time?

If you have manager permissions, go to **Team Reports** in the navigation.

### Q: Why can't I see certain projects?

You can only see projects assigned to you by your administrator.

### Q: Can I delete an entry?

Yes, go to **Time Entries**, find the entry, and click the **Delete** button.
